Es handelt sich um eine C# Applikation.
Die software is in Visual Studio 2013 (Express Edition fuer Desktop) geschrieben.

Programausfuehrung
Es gibt ein kleines demo Program, das von der Kommandozeile aufgerufen wird.
Das Projekt dazu heisst VendingMachineApp.
Komplieren und dann VendingMachineApp.exe aufrufen.

Testausfuehrung
Die unit tests (im Projekt VendingMachineTests) kann man auf 2 Arten ausfuehren.
1. Durch Visual Studio 2013 (Express Edition, Desktop).
2. Ueber die Kommandozeile mit mstest (aber sieht so aus als ob die express Version nicht mstest hat).
